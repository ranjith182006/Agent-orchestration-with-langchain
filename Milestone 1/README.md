# AI Chatbot

This is a simple AI chatbot project built using Python.

## Features

* Answer simple questions like "What is AI?" and "What is Computer?"
* Uses a local AI model
* Command line chatbot interface

## Technologies Used

* Python
* Transformers
* Torch

## Installation

1. Clone the repository

git clone https://github.com/ranjith182006/AI-chatbot.git

2. Install the required libraries

pip install -r requirements.txt

3. Run the chatbot

python chatbot.py

## Example

User: What is AI?
Bot: Artificial Intelligence is the simulation of human intelligence in machines.
